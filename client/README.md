# CodeConnect

A full-stack MERN developer networking platform that enables developers to showcase projects, connect with peers, and discover collaboration opportunities.

## Features

- JWT Authentication & Protected Routes
- Create, Update and Manage Projects
- Personalized Dashboard
- Developer Profiles
- Responsive UI with React & Tailwind CSS

## Tech Stack

- Frontend: React, Tailwind CSS, Axios
- Backend: Node.js, Express.js
- Database: MongoDB, Mongoose
- Authentication: JWT

## Setup

```bash
# Backend
cd server
npm install
npm run dev

# Frontend
cd client
npm install
npm run dev
```
